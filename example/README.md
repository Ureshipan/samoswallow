# samoswallow demo app

Минимальное показательное приложение для samoswallow. Один файл на Python без
зависимостей — чтобы наглядно потрогать все функции платформы.

## Как запустить в самосвале

1. Залей содержимое этой папки в отдельный GitHub-репозиторий (важно, чтобы
   `swallow.yaml` и `Dockerfile` лежали в корне репозитория).
2. В веб-интерфейсе самосвала: **Новое приложение** → укажи URL репозитория,
   имя `demo`, поддомен `demo`, ветку (`main` или `master`).
3. Нажми **Deploy**. Самосвал склонирует репу, соберёт образ и поднимет инстанс.
4. Открой `http://demo.<твой-домен>` — там дашборд с кнопками.

## Что показывает

| Функция самосвала | Как пощупать |
|-------------------|--------------|
| Переменные среды  | Кнопка **Env** показывает `GREETING`. Поменяй её и сделай redeploy — текст на странице обновится. |
| Нагрузка (CPU)    | Кнопка **CPU нагрузка** — смотри, как растёт CPU% в карточке инстанса. |
| Нагрузка (RAM)    | Кнопка **Занять +50 МБ** — смотри рост RAM (лимит 128 МБ из `swallow.yaml`). |
| Healthcheck       | `swallow.yaml` пингует `/health`. |
| Логи              | Каждый запрос пишется в stdout — виден в логах инстанса. |
| Авто-рестарт      | Кнопка **💥 Crash** роняет процесс; самосвал перезапускает инстанс. |
| Несколько инстансов | Подними ещё инстансы и сравни `hostname` на дашборде — он у каждого свой. |

## Локальный прогон (без самосвала)

```sh
docker build -t demo .
docker run --rm -p 8080:8080 -e GREETING="Привет!" demo
# открой http://localhost:8080
```
